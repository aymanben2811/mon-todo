# Mon Todo App

## Description du projet
Application de todo list avec authentification utilisateur.

## Stack technique
- **Next.js 14** — framework React avec App Router
- **TypeScript** — typage statique
- **Tailwind CSS** — styles utilitaires
- **Supabase** — base de données PostgreSQL + authentification

## Conventions de code
- Composants React : PascalCase (ex. `TodoItem`, `AuthForm`)
- Fonctions et variables : camelCase (ex. `fetchTodos`, `isLoading`)
- Commentaires : en français uniquement
- Ne jamais modifier les fichiers dans `node_modules/`
